# Mega-Assembler

Complete disassembly of the MSX cartridge "Mega-Assembler", a 
resident Z80 assembler / monitor / editor.

This source produces a bit-perfect binary of the original 16 KB ROM. The
MD5 of the assembled output matches `ae4cae2a812e18399eafc254701df270`.

Compile using [sjasmplus](https://github.com/z00m128/sjasmplus).
